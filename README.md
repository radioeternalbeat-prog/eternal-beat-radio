# Eternal Beat Radio Chile 🎶

Radio online 24/7 de Progressive House, Melodic Techno y Minimal desde Chile.

## Características

- **Stream 24/7** — Transmisión continua en alta calidad
- **On Demand** — Biblioteca de tracks y playlists organizadas
- **Chat en Vivo** — Interacción en tiempo real con la comunidad
- **Programación** — Calendario de shows y DJs en vivo
- **DJ Live** — Transmisión en vivo con BUTT
- **iTunes Import** — Importa tu biblioteca desde iTunes/Music
- **PWA** — Instalable como app nativa en cualquier dispositivo

## Tech Stack

- React 18 + Vite
- React Router v6
- Lucide React Icons
- VitePWA
- AzuraCast (streaming backend)

## Setup

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
